    Summary: Latitude vs. X

    The purpose of this project was to analyze hot weather changes as you get closer to the equator. This was accomplised by pulling data from the OpenWeatherMap API and creating a dataset of at least 500 cities.

    The dataset was the plotted using Matplotlib to assemble various graphs of weather vs. latitude, as picture below.
